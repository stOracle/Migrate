To run, open the group_22_assignment_5.pyde file, ensure Processing is in Python mode, and hit "Run".

INTERACTIVITY INSTRUCTIONS:
------------------------------------------
KEYBOARD:
1. If the user presses "P", the simulation will pause until they hit it again.
2. If the user presses a number between 0 and 9, a sphere will spawn into the simulation
   with an initial velocity. The mass will scale based on the number entered.
3. If the user presses "R", the simulation will completely restart.

CAMERA CONTROL:
4. If the user holds down a mouse button, the location of the camera will change based on where he/she moves the mouse.
